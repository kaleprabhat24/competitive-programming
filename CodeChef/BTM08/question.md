## Problem Statement

### The XOR Operator

The Bitwise **XOR** operator - AAA ⊕\oplus⊕ BBB === CCC. 

iii-th bit of CCC will be set (be equal to 111) if and only if the iii-th bit of AAA and BBB are different (one is 111 and other is 000). 

The xor operator is represent as `^` in programming code.

Alternatively you can view the XOR operator as - if the iii-th bit of BBB is 111 then, the iii-bit of the output will be iii-th bit of AAA flipped, else it will be same as the iii-th bit of AAA.

Flipping a bit means to change the bit from 111 to 000 or 000 to 111.

**Example**

```
      11001000  (200 in decimal)
    ^ 10111000  (184 in decimal)
      -------- 
    = 01110000  (112 in decimal)

```
