## Problem Statement

### Fibonacci Series

Given an integer NNN, output the NthN^{th}Nth term in the Fibonacci Series.

The Fibonacci Series is: `0 1 1 2 3 5 8...`

Complete the function `fib()` function to solve this problem using **recursion**.

### Input Format

- The input contains a single integer NNN.

### Output Format

Output the NthN^{th}Nth term in the Fibonacci Series.

### Constraints

- 1≤N≤1001 \leq N \leq 1001≤N≤100
