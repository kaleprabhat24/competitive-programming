## Problem Statement

### Vacation Clothes

Chef is going on a vacation for NNN days. Each day, he must wear a fresh set of clothes.

Whenever a set is worn on day ddd, it is sent for cleaning and becomes available again only on day d+7d + 7d+7. A set that returns from cleaning on some day may be worn again that same day.

What is the minimum number of sets of clothes Chef must carry so that he always has something to wear for all NNN days?

### Input Format

- The first and only line contains a single integer NNN.

### Output Format

Output the minimum number of sets of clothes Chef needs to carry.

### Constraints

- 1≤N≤101 \le N \le 101≤N≤10

### Sample 1:

Input

Output

```
3

```

```
3

```

### Explanation:

Chef needs to carry 333 sets of clothes for each of the 333 days.

### Sample 2:

Input

Output

```
10

```

```
7

```

### Explanation:

Chef only needs to carry 777 sets, because he can wear the first set again on day 888, the second set again on day 999 and the third set again on day 101010.
